# ExchangeRateMonitor

## Описание

**ExchangeRateMonitor** — это Python-скрипт для мониторинга курсов валют (USD, EUR, RUB, CNY) через API Национального банка Республики Беларусь. Скрипт проверяет обновления курсов каждые 5-10 минут в рабочие часы (с 9:00 до 16:00) и фиксирует изменения в лог-файл.

## Функциональность

- Получение текущих курсов валют с API НБ РБ.
- Сохранение текущих курсов в JSON-файл.
- Проверка обновления курсов с интервалом 10 минут.
- Запись изменений курсов валют в лог-файл с указанием времени изменения.
- Запуск мониторинга в рабочие часы (с 9:00 до 16:00).

## Установка и запуск

### 1. Установка зависимостей

```sh
pip install -r requirements.txt
```

### 2. Запуск мониторинга

```sh
python exchange_rate_monitor.py
```

## Файлы проекта

- exchange_rate_monitor.py — основной скрипт для мониторинга курсов валют.
- test_exchange_rate_monitor.py — тесты для проверки работы функций.
- exchange_rates.json — файл для хранения последнего зафиксированного курса.
- exchange_rate_changes.log — лог-файл с изменениями курсов валют.
- README.md — документация проекта.

## Тестирование

Для запуска тестов выполните команду:

```sh
python -m unittest test_exchange_rate_monitor.py
```

## API

Скрипт использует публичное API Национального банка Республики Беларусь: https://api.nbrb.by/exrates/currencies
